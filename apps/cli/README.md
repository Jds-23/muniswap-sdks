# uni-v4-script

CLI for Uniswap V4 scripts.

## Setup

```bash
pnpm install
cp .env.example .env
# edit .env with your PRIVATE_KEY and RPC_URL
```

## Usage

```bash
pnpm build
pnpm start wallet  # show address + balances on Arbitrum
```

## Dev

```bash
pnpm dev      # watch mode
pnpm lint     # check
pnpm format   # fix formatting
```
